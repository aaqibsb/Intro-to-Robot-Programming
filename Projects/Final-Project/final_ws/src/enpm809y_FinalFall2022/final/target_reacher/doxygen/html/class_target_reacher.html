Detailed Description

ROS2 node for reaching a target location using odometry data and aruco markers.

This node subscribes to the "/aruco_markers" and "/goal_reached" topics, and publishes geometry_msgs::msg::Twist messages to the "/robot1/cmd_vel" topic. It uses the data from the "/aruco_markers" topic to determine the final destination origin from final_params.yaml file, and the geometry_msgs::msg::Twist messages to control the robot's movement. The node also broadcasts the final destination of the robot as a static transform.

Constructs a new TargetReacher object.

Initializes the node and sets up subscriptions to the "/aruco_markers" and "/goal_reached" topics, as well as a publisher for the "/robot1/cmd_vel" topic. It also reads in various parameters from the parameter server, including the target location, the final destination, and the frame IDs for the various transforms. A wall timer is set up to call the cb2 function every second.
